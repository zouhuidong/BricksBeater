## 想自制关卡？

这得用文本编辑器手动画！我暂时没做工具。

如果真的想做，可以参考 ./res/level/ 内的关卡文件（关卡文件 level_xxx.dat 直接用文本编辑器打开即可），以及项目目录下的 readme.txt。关卡地图就是以字符画的形式存储的，相信你一看就知道该怎么做了。

注意，所有关卡地图的长宽必须一致，它们定义在 ./BricksBeater.ini 中。如果新增了关卡，也要在 ./BricksBeater.ini 中修改关卡数。
